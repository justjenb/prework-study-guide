HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The <section> element aids with separating content into categories.
  • A margin indicates how much space we want around the outside of anelement.
  • A padding indicates how much space we want around the content inside an element.
  • A border is a solid border surrounding the HTML component.
  • You can assign colors using semantic references or you can use its hex #.
  • Add links using '<a href="url">description</a>'.
  • Add extra bullets for more context by creating another <ul> element under your current list item.

CSS

  • CSS assists with adding style and decorative HTML elements to a web page.
  • Using class selectors allows HTML elements to be assigned to specific components on a web page in spite of global settings.
  • To designate a class selector, you put '.' in front of the selector name in your style.css file.
  • The box-shadow property adds shadow effects around an elements frame.
  • You can import custom fonts using the '@font-face' element if you source your fonts and build a package with ChunkFive Font Generator.
    • You can switch between multiple font versions by specifying the font-weight and font-style within the '@font-face' element.

Git

  • Git allows developers to work in tandem on the same code and ensure compatibility with production branches with the use of pull requests and reviews.
  • Always make sure you are working from the main repo when creating a new feature request branch or pushing changes by using the 'git checkout main' command.
  • Create a new feature branch with 'git checkout -b feature/branch-name'.
  • Push changes to a the new branch by using 'git push origin feature/branch-name'.
  • When making commits, use as concise text as possible.
  • Use 'git add -A' to add all changes to your commit.
  • To ensure your repo is up to date use 'git pull'.
  • To see the status of your repo use 'git status'.

JavaScript

  • JavaScript is used to add logic to websites with the use of scripts on the web server.
