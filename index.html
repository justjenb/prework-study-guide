Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The <section> element aids with separating content into categories.

CSS

  • CSS assists with adding style and decorative elements to a web page.

Git

  • Git allows developers to work in tandem on the same code and ensure compatibility with production branches with the use of pull requests and reviews.
  • Always make sure you are working from the main repo when creating a new feature request branch or pushing changes by using the 'git checkout main' command.
  • Create a new feature branch with 'git checkout -b feature/branch-name'
  • Push changes to a the new branch by using 'git push origin feature/branch-name'
  • When making commits, use as concise text as possible.
  • Use 'git add -A' to add all changes to your commit.
  • To ensure your repo is up to date use 'git pull'.
  • To see the status of your repo use 'git status'.

JavaScript

  • JavaScript is used to add logic to websites with the use of scripts on the web server.
